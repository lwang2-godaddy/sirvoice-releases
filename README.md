# SirVoice Releases

Download the latest version of SirVoice for macOS.

## Install

**Direct Download:** Go to [Releases](https://github.com/lwang2-godaddy/sirvoice-releases/releases/latest) and download the .dmg file.

**Homebrew:**
```bash
brew tap lwang2-godaddy/sirvoice
brew install --cask sirvoice
```

## About

SirVoice is a system-wide voice dictation app for macOS powered by AI. Press Option+Space anywhere to dictate, and AI-transcribed text appears in the focused text field.

- Website: https://sircharge.online
- Requires macOS 14 Sonoma or later
